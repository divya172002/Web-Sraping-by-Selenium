Amazon Web Scraper

A simple Python project to scrape Amazon product data using Selenium, BeautifulSoup, and Pandas. The script collects product details and saves them into a excel file.

Features

Scrapes product title, price and link

Uses Selenium to load dynamic pages

Parses HTML with BeautifulSoup

Exports clean data using Pandas

Tech Used

Python

Selenium

BeautifulSoup (bs4)

Pandas
